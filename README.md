![YAY](./assets/yay.png)

> [!NOTE]
>  Only supports MacOS (Apple Silicon) for now, other platforms and operating systems planned for the future.

## Arguments

```sh
# Open TUI interface
yay
```

```sh
# Start background daemon
yay start
```

```sh
# Stop background daemon
yay stop
```

```sh
# Display current version
yay version
```

```sh
# Display help message
yay help
```
